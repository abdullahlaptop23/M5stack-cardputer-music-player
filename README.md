M5Stack Cardputer – Audio Player Firmware
This project is a custom audio player firmware designed for the M5Stack Cardputer. It provides a retro‑style music visualizer, track information display, and control interface.
Audio output is handled through the UNIT Audio Player module.

🎵 Features
Full compatibility with UNIT Audio Player

Retro‑style audio visualizer

Track info: name, duration, track number

Play / Pause control

Clean and fast UI

Lightweight and optimized firmware

📦 Requirements
M5Stack Cardputer

UNIT Audio Player

microSD card (for audio files)

Arduino IDE or PlatformIO

🔧 Installation
Clone the repository: https://github.com/abdullahlaptop23/M5stack-cardputer-music-player
Open the project in Arduino IDE or PlatformIO.

Install required M5Stack libraries.

Upload the firmware to the Cardputer.

Connect the UNIT Audio Player and insert your microSD card.

▶️ Usage
The audio player UI starts automatically on boot.

On-screen controls:

A: Volume

B: Goto / Track select

Y: Visualizer

I: Info

Play/Pause can be controlled via the top buttons.
